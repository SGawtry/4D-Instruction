Farm & Field test list

This is a table describing the test options you can select when entering new Farm & Field order forms.

Field name Test name and related notes
Reg Regular test
S Sulfur
Zn Defunct zinc only – do not use
ZC Zinc, Copper, Iron, Manganese
B Boron
CM Calcium & Magnesium
Or Defunct organic matter only – do not use
SS Soluble salts
NO3 Nitrate. Select "1" for surface nitrate (0-6"). If they ordered split nitrate and no other special tests, select "2", and use the second special number in the "Sm#" field. For 0-24" nitrate, enter "1" but then delete the 0-6" test and replace it with the 0-24" test later through the order menu. See Stephanie for other situations or questions.
