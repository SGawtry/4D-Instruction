Create a PDF for a Soil Report

Using 4D, generate a PDF of one specific soil test report.

  1. Click the order to highlight it.
  2. In the top drop-down menu, select Order > Report Test Results.
  3. Uncheck the marked boxes for "Laboratory", "Extension", and "Firm/Agency" and click Print.
  4. Change the printer name to "Adobe PDF". Click OK twice.
  5. When a third window pops up, click OK. You do not need to load any paper.
  6. In the resulting "Save As.." window, navigate to STRAL > Shared Ofc 1&2 > STL Email Reports > (Current Year).
  7. Name the file as the order number followed by the client's last name or company name (e.g. 71452 Johnson). Click Save.
    Adobe Acrobat DC will automatically open the PDF.
  8. Proceed to Edit a Soil Report in Adobe Acrobat
